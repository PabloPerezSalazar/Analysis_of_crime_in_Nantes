Analyse de l'évolution de la criminalité à Nantes¶

Objectifs de l'étude :¶

  1. Quels sont les crimes les plus courants.
  2. Savoir comment la criminalit a volu ces dernires annes Nantes.
  3. Si la tendance se poursuit, quels sont les crimes qui connatront la plus forte croissance l'avenir.
  4. Est-ce qu'il a une correlation entre le crime et le chomage ?

def dataframe_dict(multi_sheet_file):
    """
    Create one Dataframe for each sheet in a Excel file.
    """
    # Gets the sheet names in the file
    excel_sheet_names = multi_sheet_file.sheet_names

    dict_of_sheets = {}
    for sheet in excel_sheet_names:
        dict_of_sheets["sheet_" + sheet] = pd.read_excel(multi_sheet_file, sheet_name=sheet, header = 1)
         #  header = 1 for use second row as a name of columns
    
    return dict_of_sheets

    """
    This is where the altair chart is defined, using the pandas dataframe.

        vols.mark_point()
        + vols.transform_loess("Annee", "Delits enregistres").mark_line(size=0.5).encode().interactive() #ligne de lissage

Prévisions pour chaque délit¶

In [32]:
def altair_visualization_predictions(police_records_agregat):
    """
    This is where the altair chart is defined, using the pandas dataframe.

    :returns: altair chart
    
    https://altair-viz.github.io/user_guide/compound_charts.html
    """
    charts = []
    for delit in police_records_agregat['Type de crime'].unique() :        
        police_records_filtre = police_records_agregat[police_records_agregat['Type de crime'].str.contains(delit)]
        
        with warnings.catch_warnings():
            warnings.simplefilter('ignore')
            police_records_filtre['Annee'] = police_records_filtre['Annee'].astype(str)\
                            .apply(lambda x: dt.datetime.strptime(x,'%Y'))

            police_records_ari = police_records_filtre
            # datatime as index
            police_records_ari.index = police_records_ari['Annee']
            police_records_ari = police_records_ari['Delits enregistres']

            # modlisation et entrainement
            mod = sm.tsa.statespace.SARIMAX(police_records_ari,trend='c', order=(1,0,0)) # (p,d,q) pour un modle ARIMA non saisonnier

            results = mod.fit(disp=False)

            df_fcast = pd.DataFrame(results.forecast(steps=3))
            df_fcast["date"] = df_fcast.index
            df_fcast = df_fcast.rename(columns={0: "delits"}).reset_index(drop=True)

            prediction = alt.Chart(df_fcast).mark_bar(color="orange").encode(
            x='date:T',
            y='delits:Q'
            )

            visualisation = alt.Chart(police_records_filtre.reset_index(drop=True)).encode(
                alt.X("Annee", axis=alt.Axis(title="", grid=False)),
                alt.Y("Delits enregistres:Q", axis=alt.Axis(title="Nombre de delits enregistres", grid=False)),
                ).properties(
                    title="Estimation de l'volution des " + delit + " dans les annes  venir",
                    width=500,
                    height=300,
                )
            
            charts.append(alt.layer(
                prediction, 
                prediction.transform_loess("date", "delits").mark_line(color =  'orange', size=0.5).encode().interactive(), 
                visualisation.mark_bar(),
                visualisation.transform_loess("Annee", "Delits enregistres").mark_line(size=0.5).encode().interactive()
            ))
            
    return charts

Nous pouvons constater qu'une baisse du chmage entrane une baisse de la criminalit, mais celle-ci n'est pas immdiate.

En d'autres termes, nous constatons que si le chmage augmente, la criminalit augmente (2012-2015), mais apparament si le chmage diminue, la criminalit semble mettre du temps baisser, mais elle diminue aussi. La tendance 2019-2020 confirme ce comportement.
